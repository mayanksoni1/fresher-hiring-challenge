
# Travel.Com – Tour & Travel Project

This is my **major project** showcasing a dynamic travel website.  
It is built using **HTML, CSS, JavaScript (frontend)** and **Node.js + Express + MongoDB (backend)**.  

## 🚀 Features
- Dynamic destinations (stored in MongoDB)
- Search & filter by location, budget, and type
- Booking system with confirmation
- AI chatbot (Botpress Cloud) for FAQs
- Google Maps integration for destinations
- User authentication with favorites

## 📂 Project Structure
- `frontend/` → HTML, CSS, JS files
- `backend/` → Node.js + Express server
- `README.md` → Documentation

## ▶️ How to Run
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start backend: `node server.js`.
4. Open `main.html` in browser.
